Description du projet

Le projet Azure resume à pour but de me familiariser avec l'environnement Azure dans un contexte de projet et également de montrer mes compétences grâce à ce site web. Ce site web à pour but d'afficher mon cv et les futures projets que je réaliserai.
En plus cela, le site peut comptabiliser le nombre de visiteurs grâce au code javascript et au backend mis en place.

Concernant l'infrastructure, le projet se réparti en 3 parties :
- La partie Frontend : Le code HTML, CSS et Javascript
- La partie Backend : Blob storage, CDN, l'API Azure function en python et la cosmosdb table
- La partie CI/CD : Git et Github action

Le site web est un site web statique comprenant du code HTML, CSS, javascript. Le contenu du site web est stocké dans un blob storage. Il est ensuite affiché grâce à un CDN.
Le but principal du code js est de faire une requête HTTP en direction de l'API azure function à chaque fois que quelqu'un visite le site.

L'azure function, est une fonction serverless codée en python qui se déclenche à la réception d'une requête HTTP. La fonction va augmenter le nombre total de visiteurs de 1 à chaque visite du site.
Le nombre total est stocké dans la cosmosdb table. Le code du site et l'azure function sont stockés dans github. Le déploiement des mises à jours de ces éléments se font automatiquement grâce à github actions.
